# Audio Visualizer

Audio Visualizer is a simple open-source tool to visualize the audio generated by your microphone

## Installation

Use the package manager pip or your own package manager to install requirements.txt

```bash
pip install -r requirements.txt
```
```bash
apt install python3-numpy python3-pyaudio python3-pygame
```

## Usage

```python
python3 audiovisualizer.py
```
